# 三. 新闻爬虫案例

# 1. 案例目标

# 业务目标

  1. 从 http://news.163.com/ (opens new window) 抓取 10 条「热点排行」数据，包括：新闻标题和新闻 URL；

  2. 将新闻标题和新闻 URL 保存至 /data/news.json 文件中，数据格式如下：

    [
  {
    time: NewsInfo[],
  },
]

# 技术方案

  1. 使用 TypeScript 实现一个终端应用；
  2. 使用 ts-node 编译并执行 TypeScript；
  3. 使用 axios 实现网络请求；
  4. 使用 iconv-lite 实现 GBK 编解码；
  5. 使用 cheerio 实现页面节点解析。

# 2. 新建项目及准备

  1. 新建目录 news-spider；

  2. 初始化项目：

    npm init -y

  3. 安装开发依赖包：

    npm i typescript ts-node -D

  4. 初始化 TypeScript 配置，生成 tsconfig.ini：

    npx tsc --init

  5. 新建 src/index.ts 作为系统入口，先输入以下代码：

    console.log('news spider')

  6. 修改 package.json 配置开发脚本和构架脚本，代码如下：

    "main": "src/index.ts",
"scripts": {
  "dev": "ts-node src/index.ts",
  "build": "tsc"
},

  7. 执行 npm run dev 测试运行，确保控制台能够正确输出 news spider；

  8. 修改 tsconfig.json 设置 outDir 指定编译结果的目标目录，代码如下：

    "outDir": "./dist",

  9. 执行 npm run build 确保能够在 dist 目录下编译生成 index.js 文件。

# 3. 爬虫业务结构

# 4. 爬虫业务实现

# 4.1 获取网页 HTML

  1. 安装开发包：

    npm i axios cheerio iconv-lite

  2. 在 src/index.ts 中输入以下代码：

    /**
 * 网易新闻排行爬虫
 */
class TopNewsSpider {
  /// 爬取网站地址
  private url = 'http://news.163.com/'

  /** 开始爬取工作 */
  async startSpiderProcess() {
    const html = await this.getRawHTML()
    console.log(html)
  }

  /** 获取原始 HTML 内容 */
  private async getRawHTML() {}
}

const spider = new TopNewsSpider()

spider.startSpiderProcess()

  3. 引入 axios 和 iconv-lite：

    import axios from 'axios'
import iconv from 'iconv-lite'

  4. 修改 getRawHTML 实现请求网页数据并完成 GBK 解码，代码如下：

    /** 获取原始 HTML 内容 */
private async getRawHTML() {
  const { data }: { data: Buffer } = await axios.get(this.url, {
    responseType: 'arraybuffer',
  })

  return iconv.decode(data, 'gbk')
}
    • responseType 表示服务器响应的数据类型，可以是：'arraybuffer' / 'blob' / 'document' / 'json' / 'text' / 'stream'；
    • 默认响应的数据类型是：'json'。

  5. 运行 npm run dev 确认能够获取网易新闻的首页 HTML 并能够正确解码（控制台输出没有乱码）。

# 4.2 使用 cheerio 解析 HTML

  1. 引入 cheerio：

    import cheerio from 'cheerio'

    WARNING

    由于 cheerio 是用 JavaScript 开发的，要在 TypeScript 中使用 cheerio，还需要根据系统提示安装一个类型定义文件（*.d.ts）—— @types/cheerio，命令如下：

    npm install @types/cheerio -D

  2. 新建 getTopNews 方法，负责从 HTML 中读取「热点排行」列表，代码如下：

    /** 开始爬取工作 */
async startSpiderProcess() {
  const html = await this.getRawHTML()

  this.getTopNews(html)
}

/** 获取新闻排行列表 */
private getTopNews(html: string) {
  const $ = cheerio.load(html)

  const list = $('.mod_hot_rank ul li')

  list.map((idx, el) => {
    const title = $(el).find('a').text()
    const url = $(el).find('a').attr('href') ?? ''
    const count = parseInt($(el).find('span').text())

    console.log(title, url, count)
  })
}
    • 运行 npm run dev 确认能够在控制台输出 10 条热点新闻标题、URL 和点击数量。

  3. 在 index.ts 中定义新闻信息接口和新闻数据接口，代码如下：

    /** 新闻信息接口 */
interface NewsInfo {
  title: string
  url: string
  count: number
}

/** 新闻数据接口（记录爬虫返回结果） */
interface NewsData {
  time: number
  data: NewsInfo[]
}

  4. 继续修改 getTopNews 方法，生成并返回抓取时间及热点新闻数据，代码如下：

    /** 获取新闻排行列表 */
private getTopNews(html: string): NewsData {
  const $ = cheerio.load(html)

  const list = $('.mod_hot_rank ul li')
  const news: NewsInfo[] = []            // 新增

  list.map((idx, el) => {
    const title = $(el).find('a').text()
    const url = $(el).find('a').attr('href') ?? ''
    const count = parseInt($(el).find('span').text())

    news.push({ title, url, count })     // 新增
  })

  // 以下新增
  return {
    time: new Date().getTime(),
    data: news,
  }
}

# 4.3 保存热点新闻数据

  1. 定义新闻列表的 JSON 数据接口，方便后续使用，代码如下：

    /** 新闻列表的 JSON 数据接口 */
interface NewsJSONData {
  [time: number]: NewsInfo[]
}

  2. 引入 fs 和 path 核心模块，代码如下：

    import fs from 'fs'
import path from 'path'

  3. 定义私有变量 dir 和 filePath 以便将抓取的数据保存成文件，代码如下：

    /// 数据文件保存目录
private dir = path.resolve(__dirname, '../data')
/// 数据文件路径
private filePath = path.resolve(this.dir, 'news.json')

  4. 定义私有方法 generateJSONData 负责生成 JSON 数据，并修改 startSpiderProcess 中的方法调用，代码如下：

    /** 开始爬取工作 */
async startSpiderProcess() {
  const html = await this.getRawHTML()
  const data = this.getTopNews(html)
  const json = this.generateJSONData(data)

  fs.writeFileSync(this.filePath, JSON.stringify(json))
}

/** 生成 JSON 数据 */
private generateJSONData(data: NewsData): NewsJSONData {}

  5. 实现 generateJSONData 方法并运行测试，代码如下：

    /** 生成 JSON 数据 */
private generateJSONData(data: NewsData): NewsJSONData {
  let content: NewsJSONData = {}

  // 1. 判断目录是否存在，如果不存在新建目录
  if (!fs.existsSync(this.dir)) fs.mkdirSync(this.dir)

  // 2. 如果文件存在，先读取之前保存过的内容
  if (fs.existsSync(this.filePath)) {
    content = JSON.parse(fs.readFileSync(this.filePath, 'utf-8'))
  }

  // 3. 追加新的爬取内容
  content[data.time] = data.data

  return content
}
    • 运行 npm run dev 确认能够把爬取到的新闻数据正确的保存在 /data/news.json 中。
