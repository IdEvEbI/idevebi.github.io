# 一. 概述

学习一门新语言的路径如下：

  1. 建立基本认知
  2. 明确学习目标
  3. 了解基础语法
  4. 记录实现目标过程

# 1. 建立基本认知

先要明确这门语言能用在哪里？能解决什么问题？有什么特点？有什么成功案例？信息来源可以包含：

  • 官方网站 (opens new window)
  • 英文社区
    • Stack Overflow (opens new window)
    • Github (opens new window)
    • Twitter (opens new window)
  • 中文社区
    • InfoQ (opens new window)
    • 掘金 (opens new window)

# 2. 明确学习目标

确认自己在学习过程中，以及学习完能够做什么？给自己学习 TypeScript 定如下两个小目标：

  1. 做一个后台系统
    • Express
    • 用户登录接口
    • 商品数据 CRUD 接口
  2. 做一个前台系统
    • React + Hooks
    • 用户登录
    • 商品管理后台

# 3. 了解基础语法

要掌握一门的语言的基础语法，应该也只应该从官网获得，基本语法需要了解并掌握的内容如下：

  • 数据类型：知道能处理哪些类型的数据
  • 流程控制：知道分支、循环的实现方式
  • 函数：知道如何封装一段代码逻辑
  • 面向对象：知道如何封装属性和方法
  • 高阶语法：了解接口、泛型、装饰器等高阶语法及应用场景

TIP

学习基础语法不需要一次性掌握，更不需要死记硬背。

先对语法建立基本认知，然后在实践中边用边查。等做完一到两个项目后，大部分常用语法也就掌握了。

# 4. 记录实现目标过程

做笔记是一个非常重要的习惯，在学习过程中编写笔记有以下好处：

  1. 记录下我们在学习时的状态和心得以及重要知识的备忘；
  2. 把有些需要肌肉记忆的知识落地在笔记中留存，在需要时翻看能快速恢复记忆；
  3. 整理笔记本身就是对知识的结构、逻辑及脉络的梳理，是加深理解的有效手段。

TIP

记录学习笔记能鉴证我们成长的历程。
