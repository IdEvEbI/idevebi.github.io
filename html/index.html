# HTML 笔记

# 一. HTML 的作用和开发工具

# 1.1 HTML 的作用

HTML 不是一门编程语言，而是一个告诉浏览器如何组织页面的标记语言，被用来描述 Web 文档的结构和语义，

# 1.2 开发工具

  1. 编辑器 —— Visual Studio Code (opens new window)

    1. HTML 本质上就是一个文本文件，大多以 .html 或 htm 作为文件扩展名；
    2. 可以使用任何文本编辑器编写或修改 HTML 文件内容；
    3. 不过由于 VS Code 的价格免费、功能强大、速度飞快、插件丰富等特点备受前端工程师青睐。

  2. 浏览器 —— Google Chrome (opens new window)

    1. 虽然我们可以使用任何一款浏览器来查看 HTML 文件的结果；
    2. 不过由于 Chrome 提供了丰富的调试工具和插件，可以说是前端工程师开发必备的利器。

  2. 用户使用浏览器查看网页展示结果。

# 二. HTML 元素

# 2.1 元素的构成

HTML 由一系列的元素（elements）组成，这些元素可以用来包围不同部分的内容，使其以某种方式呈现或者工作。在 HTML 中一个元素可以包含一块文本、一张图片、或者一块数据等。一个典型的元素包含以下三个部分：

  1. 一个具有一些属性的开始标签
  2. 中间包裹的内容
  3. 一个结束标签

TIP

把元素放到其他元素之中，被称作元素的嵌套，在元素嵌套时要确保元素被正确的嵌套。

# 2.2 文档和脚本标签

序号 标签 描述
1 html HTML 文档的顶级元素，也被称为根元素
2 head 规定文档相关的配置信息（元数据）
3 body 文档的内容
4 script 嵌入或引用可执行脚本

# 2.3 常用标签

序号 标签 描述
1 p 段落
2 img 图像
3 a 链接
4 h1~h6 标题
5 div 内容容器
6 span 行内容器
7 ul 无序列表
8 ol 有序列表
9 li ul、ol 的列表项
10 hr 分隔线

# 2.4 表格标签

序号 标签 描述
1 table 表格
2 thead 表格的列头的行
3 tbody 表格内容
4 tfoot 表格中各列的汇总行
5 tr 表格中行
6 th 表头中的单元格
7 td 表格中的单元格

# 2.5 表单标签

序号 标签 描述
1 form 表单
2 input 输入框
3 button 按钮
4 lable 标签
5 textarea 多行纯文本编辑控件
6 datalist 数据列表
7 optgroup 选项组
8 select 选项菜单
9 option datalist，optgroup，datalist 中包含的项

# 2.6 文本修饰标签

序号 标签 描述
1 b 提醒注意 Bring Attention To，一般用粗体显示
2 strong 表示文本十分重要，一般用粗体显示
3 i 需要区分普通文本的一系列文本，一般用斜体显示
4 em 需要用户着重阅读的内容，一般用斜体显示
5 br 在文本中增加一个换行，在写诗和地址时很有用

# 2.7 内容标签

序号 标签 描述
1 main 展示文档的 body 或应用的主体部分
2 header 展示介绍性内容，通常包含一组介绍性的或是辅助导航的实用元素
3 nav 在当前文档或其他文档中提供导航链接
4 section 包含在 HTML 文档中的独立部分，没有更具体的语义
5 article 表示文档、页面、应用或网站中的独立结构
6 aside 表示一个和其余页面内容几乎无关的部分
7 footer 表示页脚
8 hgroup 代表一个段的标题，会在文档大纲中显示
9 address 提供了某个人或某个组织（等等）的联系信息

# 2.8 媒体标签

序号 标签 描述
1 audio 音频
2 video 视频
3 map 与 area 属性一起使用，定义热点区域关联超链接
4 area 仅在 map 元素内部使用，定义热点区域关联超链接

# 2.9 文档元数据

序号 标签 描述
1 title 文档的标题
2 link 与外部资源的关系，常用于链接样式表
3 style 包含文档的样式信息
4 base 所有相对 URL 的根 URL，只能有一个
5 meta 表示不能由其它 HTML 元数据相关的信息
