# 使用 VuePress 创建文档站

使用 VuePress 之前需要保证系统中已经：

  1. 安装了 Node.js (opens new window) 和 Git (opens new window)
  2. 注册有 Github (opens new window) 账号，并设置了 SSH (opens new window)。

# 1. 安装 VuePress

  1. 在终端输入以下命令，使用 create-vuepress (opens new window) 创建 VuePress 项目：

    yarn create vuepress docs

    在提示信息中，第一项模板类型选择 docs，其他的按照实际信息填写即可。

  2. 在终端输入以下命令，安装项目的依赖包并启动项目：

    yarn && yarn dev

    打开浏览器访问 http://localhost:8080 (opens new window) 可以看到一个和 VuePress (opens new window) 官网差不多的文档站点。

# 2. 配置 VuePress

# 2.1 实时更新问题

  1. 修改 package.json，在末尾增加以下内容：

    "resolutions": {
  "watchpack": "1.6.1"
}

  2. 在终端使用 yarn 命令安装相关的包。

# 2.2 侧边栏

.vuepress/config.js (opens new window) 是 vuepress 约定的配置文件，用于设置站点的标题、描述信息、导航、侧边栏等内容。

  • 侧边栏配置实例如下：

    sidebar: {
  '/tools/vuepress/': [     // 侧边栏目录，前后都要有 `/`
    {
      title: 'Vue Press',
      collapsable: false,
      sidebarDepth: 2,
      children: [''],
    }
  ],
}

    WARNING

    注意：侧边栏目录的前后都要有 /。

# 2.3 静态资源目录

可以将网站需要使用的静态资源保存在 .vuepress/public 目录下，其他文档通过根目录即可访问。
