# npm 使用笔记

npm（Node Package Manager）即 Node 包管理器，是 Node.js 默认的软件包管理系统，npm 是用 JavaScript 编写的。

npm 会随着 Node.js 自动安装，通过 npm 命令可以：

  • 下载并安装指定的模块
  • 把自己设计的模块分发到 registry

# 1. 查看版本和帮助

  1. 查询版本

    npm version

  2. 查看帮助

    npm help

npm help <command>

# 2. 项目初始化

  1. 创建新项目

    npm init

npm init -y
    • npm init 通过交互式创建 package.json 文件
    • npm init -y 使用默认配置创建 package.json 文件

  2. 默认生成的 package.json 说明如下：

    {
  "name": "demo", // 项目名称
  "version": "1.0.0", // 项目版本
  "description": "", // 项目描述
  "main": "index.js", // 入口文件
  // 项目脚本，可以使用 npm run `test` 运行对应脚本
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": [], // 关键字
  "author": "", // 作者
  "license": "ISC" // 许可协议
}

# 3. 包管理

  1. 访问 https://www.npmjs.com/ (opens new window) 可以搜索需要使用的包

  2. 访问 <package> 相关网页

    # 访问 <package> 首页
npm home <package>

# 访问 <package> issues
npm bugs <package>

# 访问 <package> git 仓库
npm repo <package>

  3. 安装 1 个或多个包，并保存到 dependencies

    npm i <package1name> <package2name> ...

  4. 安装 1 个或多个包，并保存到 devDependencies

    npm i <package1name> <package2name> ... --save-dev

npm i <package1name> <package2name> ... -D

  5. 检查项目的依赖包是否更新

    npm outdated

  6. 删除包

    npm rm <package>

# 4. Node.js 工具

# 4.1 全局安装和使用

  1. 全局安装，以 webpack 为例

    npm i webpack webpack-cli -g

    TIP

    全局安装默认会把包安装到 /usr/local/lib/node_modules/ 目录。

    • 安装之后就可以像使用其他终端命令一样使用这些工具。

  2. 使用（全局安装的工具可以直接使用）

    webpack index.js

  3. 卸载（卸载全局工具需要有 -g 参数）

    npm rm webpack webpack-cli -g

    WARNING

    大多工具是不需要使用全局安装的，这是因为：

    • 避免本机版本与某些项目使用工具版本不一致的问题；
    • 大多前端项目会使用本地安装，这样仅在项目目录下可以使用工具，而其他位置不嫩使用。

# 4.2 本地安装和使用

  1. 本地安装，以 webpack 为例

    npm i webpack webpack-cli -D

  2. 通过 npx 命令使用，npx 可以在项目目录下搜索并执行工具，示例如下：

    npx webpack index.js

  3. 通过 scripts 脚本使用，修改 package.json 如下：

    "scripts": {
  "bundle": "webpack index.js"
},
    • 在终端执行 npm run bundle 同样可以实现打包。

    TIP

    接手一个项目后，通过 package.json 的 scripts 能够快速知道如何使用项目，例如：启动、打包、测试等。

# 5. 发布 npm 包

# 5.1 账号准备

  1. 在 https://www.npmjs.com/ (opens new window) 注册账号；

  2. 使用 npm 命令登录，相关命令如下：

    # 登录，会提示输入用户名 / 密码 / 邮箱
npm login

# 查看当前登录用户
npm whoami

# 退出登录
npm logout

# 5.2 项目准备

  1. 创建项目：

    npm init -y

  2. 修改 package.json 中的 name 和 version，其中：

    1. name：模块名，需要保证唯一，npm 仓库上的模块名不允许重复；
    2. version：major.minor.patch。

  3. 新建 .npmignore 标注不需要上传的文件，作用与 .gitignore 类似；

  4. 新建 README.md 提供模块的使用说明；

# 5.3 发布和删除包

  1. 发布包

    npm publish

  2. 删除包

    # 删除指定版本
npm unpublish 包名@版本

# 删除整个包
npm unpublish 包名 --force

  3. 更新版本，准备重新发布新版时使用

    npm version major.minor.patch
