# 一. 课程简介

# 1. 什么是 Node.js

  1. 官网的介绍

    1. Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境；
    2. Node.js 使用了一个事件驱动、非阻塞式的 I/O 模型，使其轻量又高效。

  2. 在 Nodes.js 里运行 JavaScript 跟在 Chrome 中运行 JavaScript 有什么不同？

    • Node.js 中没有浏览器的 API，例如：document、window 等；
    • Node.js 新增了许多 API，例如：文件系统、进程等；
    • 除以上两点外，在 Node.js 和 Chrome 中编写的 JavaScript 几乎没有什么区别，因为 Node.js 和 Chrome 使用的是相同的引擎和模型。

TIP

对于开发者而言：

  • 在 Chrome 里写 JavaScript 可以控制浏览器；
  • 在 Node.js 里写 JavaScript 能够用类似的方式控制整个计算机。

# 2. Node.js 能做什么

  1. Web 服务

    • 服务端渲染（SSR）= 搜索引擎优化 + 首屏速度优化；
    • Node.js = 服务端渲染 + 前后端同构；

  2. 自动化和构建工作流

    • 目前流行的构建工具有：webpack、gulp 等；
    • 使用 Node.js 做 JavaScript 构建工具是最保险的选择；

  3. 可扩展性

    • 大型应用需要给使用者自定义模块的能力，例如：VSCode；
      • 利用 JavaScript 的灵活性能够提供外部扩展；
      • 利用 JavaScript 庞大的开发者基数，让扩展存在更多可能；
    • 在已有网站的情况下需要开发客户端 App，例如：twitch.tv；
      • 可以用 Node.js 客户端技术（electron）最大限度地复用现有工程。

# 3. 课程实战项目介绍

  • 项目目标：Node.js 版的极客时间网站。
  • 业务功能：
    • 列表页
      • 打通前后台 + 服务端渲染
    • 详情页
      • 网页路由 + 异步加载
    • 播放页
      • API 服务器（评论系统）
